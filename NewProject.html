Create a New Maven Project in IntelliJ from Instructional Video


Create the Project

In IntelliJ from welcome screen or under FILE select "new Project". Choose "Maven", then, "from archetype" and "Maven webapp".



Give your project a descriptive name that tells you what it is and choose Next a couple of times.



Copy in dependencies from another POM



Add needed directories for java, resources, and test

Select the main directory in the project view



Right click and choose "New" and then Directory. Name the directory "java" all lowercase! Otherwise, Maven won't know what to do with it. You can choose refactor and rename to fix this if you get it wrong.

Right click on the new java directory and choose "mark Directory" and then "sources Root". This tells Maven to look here for things to compile.

Right click on the java directory again and choose "New" and then package. Give it a name like "entity" or "persistence". Repeat as needed to build out your package structure.

Right click on the src directory and follow the same steps to create a new directory called "test". Under it create a directory called "java". Follow the same instructions to mark it, but mark it as "test Sources Root" instead of just "sources Root". This will tell Maven where to find the unit tests.

Finally, you can create a resources directory under test and mark it as test resources.
